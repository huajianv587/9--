# Verified Transfer-Ladder Screen

Date checked: 2026-06-01
Project: Analyst coverage and future financial stress

## Bottom Line

The 70-80% target is defensible only as a project-level ladder target, not as an AEL, APJFS, or Economic Record single-journal target. The current public screen supports:

1. **APJFS first submission** if the finance-journal strategy is preferred and the user accepts that APJFS has no verified 70-80% single-journal evidence.
2. **Economic Record transfer version** if APJFS rejects without a fatal data flaw and the paper is moved to a regular SSCI/JCR Q3 economics backup.
3. **AEL / Applied Economics fallback versions** if the user chooses a compressed letter or a broader applied-economics transfer route.
4. **Do not force a nominal Q3 journal** when scope fit or acceptance probability is poor.

The current ladder separates journal fit from probability. APJFS remains the strongest finance-journal fit. Economic Record is now the clearest full-paper **regular SSCI/JCR Q3 economics** backup found so far, with a 37% visible acceptance-rate signal and a plausible Australia/Asia-Pacific applied-economics fit. Applied Economics remains useful as a probability-first transfer outlet because it has a 40% displayed acceptance rate, but it is not Q3 in the current displayed Taylor & Francis best-quartile screen and should not be used as the strict Q3 substitute.

## Probability Discipline

The paper should not claim a mechanical probability such as:

> AEL 35% plus Applied Economics 40% equals guaranteed 61%.

Acceptance probabilities are not independent, and desk-rejection reasons carry across outlets. The practical interpretation is:

- APJFS alone: no verified 70-80% single-journal evidence, even though it remains the best finance-journal fit.
- Economic Record alone: 37% visible acceptance-rate signal, so it is a useful regular SSCI/JCR Q3 backup but not a 70-80% route.
- AEL alone: official displayed acceptance is 35%, so it is not a 70-80% route.
- Applied Economics alone: 40% displayed acceptance rate, useful as a broader empirical transfer but not the strict Q3 solution.
- APJFS plus Economic Record plus AEL/Applied Economics provides a stronger project-level ladder than any single outlet, but it still must not be described as guaranteed 70-80%.

At the current public-screen stage, **do not add another journal just to make the arithmetic look better**. A target should be added only after current JCR/Master Journal List verification and scope review.

## Transfer Decision Rules

If APJFS rejects with "not enough finance contribution" but no fatal data objection:

- Transfer to Economic Record.
- Use the Economic Record manuscript that foregrounds Australia, Singapore, market information, and applied-economics relevance.
- Keep the causal boundary explicit.
- Do not describe Economic Record as a 70-80% journal.

If AEL rejects with "too narrow" or "not enough novelty":

- Transfer to Applied Economics.
- Expand the manuscript to a fuller applied-economics article.
- Keep the title focused on analyst coverage and future financial stress.
- Add a fuller literature and data-construction discussion.
- Keep disagreement measures secondary.

If AEL rejects because the stress label is not credible:

- Do not immediately transfer.
- Add the conservative label-component robustness explanation more prominently.
- Consider a strict-label sensitivity only if the reviewer explicitly demands it.

If AEL rejects because the paper is outside economics:

- Applied Economics is still plausible if the introduction emphasizes market information environment and applied empirical economics.
- Finance-only journals should be used only after checking acceptance rate and SSCI status; current screen does not show a stability-first finance-only target.

## Current No-APC Instruction

For Taylor & Francis hybrid journals in this ladder, use the standard subscription publication route. Do not select paid Open Select / gold open access unless an institutional agreement or waiver is confirmed. For Wiley hybrid outlets such as Economic Record, do not select paid open access unless the portal requires a choice and the user explicitly approves it or confirms institutional coverage.
