Candidate outlet type:

- SSCI finance or business-finance Q3 journal.

Use if AEL rejection says:

- analyst mechanism is weak;
- broad stress label needs more accounting explanation.

Action:

- Emphasize financial-stress screening and accounting robustness.
- Treat analyst coverage as one information variable.
- Add a supplemental strict-label or component sensitivity table if available.

Candidate outlet type:

- SSCI accounting, risk, or applied finance journal with Q3 status verified in current JCR.
