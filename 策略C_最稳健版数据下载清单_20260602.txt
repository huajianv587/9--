# 策略C：最稳健版数据下载清单

日期：2026-06-02
项目：Analyst Coverage and Future Financial Stress
目标：正规 SSCI/SCIE 同级别 Q3，尽量提高稳健性和接收概率

---

# 一、核心原则

本项目如果走“最稳健策略C”，数据下载目标不是简单补几个字段，而是把论文从“会计压力关联研究”升级为：

1. 有真实或准真实困境事件标签；
2. 有完整或接近完整的 Altman-style distress 指标；
3. 有分析师覆盖、覆盖变化、预测分歧、预测修正等信息环境变量；
4. 有市场变量控制可见度、流动性和市场风险；
5. 尽量包含 inactive / delisted firms，避免幸存者偏误；
6. 所有预测变量必须发生在 outcome 之前，避免 look-ahead bias。

---

# 二、样本下载范围

## 1. 市场范围

最低版本：
- Australia / ASX
- Singapore / SGX / Catalist

如果数据可得、时间允许，建议扩展到：
- Hong Kong / HKEX
- Malaysia / Bursa Malaysia
- Thailand / SET
- Korea / KRX
- Japan / TSE

但是注意：
如果扩展市场会拖慢项目，先不要扩太多。稳三区优先不是市场越多越好，而是标签和识别越清楚越好。

推荐顺序：
1. Australia + Singapore，当前主样本；
2. Hong Kong 作为第三市场；
3. Malaysia / Thailand 作为薄覆盖市场补充；
4. Korea / Japan 可作为稳健性，但制度差异较大。

## 2. 公司范围

必须包含：
- Active listed companies
- Inactive listed companies
- Delisted companies
- Bankrupt / liquidated / suspended / restructuring firms if available

不要只下载 active firms。只下载 active firms 会产生 survivorship bias，这是审稿人的重大攻击点。

公司类型：
- listed common equity
- ordinary shares
- operating companies

建议排除或单独标记：
- funds
- ETFs
- REITs，如果样本过多可单独控制或剔除
- SPACs
- preferred shares
- financial firms，如果使用 Altman Z-score，银行/保险应单独处理或剔除

## 3. 时间范围

推荐：2010-2024

最低：2014-2024

说明：
当前已有 2014-2024。如果可以下载，建议向前扩到 2010，增加事件数和时间跨度。

| delisting_reason | Delisting Reason | 区分困境退市和并购退市 |
| bankruptcy_filing_date | Bankruptcy Filing Date | 严格困境事件 |
| bankruptcy_status | Bankruptcy Status | 严格困境事件 |
| liquidation_date | Liquidation Date | 清算事件 |
| liquidation_status | Liquidation Status | 清算状态 |
| restructuring_date | Restructuring Date | 债务重组/困境重组 |
| default_event_date | Default Date / Debt Default Date | 违约事件 |

| suspension_reason | Trading Suspension Reason | 区分困境与其他原因 |

## 用途

构建最严格的因变量：

strict_distress_event_12m = 1，如果未来12个月或下一财年发生：
- bankruptcy
- liquidation
- receivership / administration
- default
- distressed restructuring
- distress-related delisting
- long trading suspension due to financial distress

如果真实事件数量不够，再使用 broader distress label。

# 十三、重要提醒

即使走策略C，也不能诚实承诺单一期刊70-80%接收率。
正规SSCI/SCIE Q3金融经济期刊通常没有这么高的官方接收率。

策略C的价值是：
- 显著降低拒稿风险；
- 把论文从“可投”提升为“比较稳健”；
- 让审稿人更难攻击因变量、幸存者偏误、内生性和数据透明度；
- 提高顺序投稿组合的项目级成功率。

最稳妥表达：
目标是通过数据增强和保守定位，把论文提升到正规Q3期刊可认真审理并有较高修回机会的水平，而不是保证单刊70-80%。
